ДИПЛОМНА РОБОТА
Тема: "TechCare 2025 - Сучасна система моніторингу та обслуговування комп'ютера"

ОПИС ПРОГРАМИ:
TechCare 2025 - це desktop застосунок з сучасним неоновим дизайном, написаний на Python з використанням Tkinter. Програма має інноваційний графічний інтерфейс у стилі 2025 року з неоновими акцентами та відповідає найсучаснішим стандартам UI/UX дизайну.

ОСНОВНІ ФУНКЦІЇ:

1. МОНІТОРИНГ СИСТЕМИ З НЕОНОВИМИ ЕФЕКТАМИ
   - Відстеження CPU, RAM, використання диска у реальному часі
   - Розрахунок температури процесора з градієнтними індикаторами
   - Контроль швидкості вентиляторів з анімованими віджетами
   - Час роботи системи з динамічним оновленням
   - Сітка параметрів 2x3 з неоновою обводкою #00DDEB

2. AI АНАЛІТИКА З ФУТУРИСТИЧНИМ ДИЗАЙНОМ
   - Прогнозування здоров'я системи з кольоровими індикаторами
   - Власний алгоритм аналізу без готових ML бібліотек
   - Інтерактивні рекомендації з неоновими акцентами
   - Динамічний індекс здоров'я системи
   - Анімований запуск AI аналізу

3. ДІАГНОСТИКА ТА РЕМОНТ З НЕОНОВИМИ КНОПКАМИ
   - Автоматична діагностика з кольоровими індикаторами
   - Завершення зависших процесів (CPU > 80%)
   - Очищення системних ресурсів
   - Детальні звіти з неоновими статусами
   - Кнопки з hover-ефектами

4. СИСТЕМА ДОСЯГНЕНЬ З ГЕЙМЕРСЬКИМ ДИЗАЙНОМ
   - Нарахування очок за активність (5-20 за завдання)
   - Візуальні рівні користувача з золотими акцентами
   - Різні типи досягнень з іконками
   - Мотиваційні елементи з неоновим підсвічуванням

5. ПЛАНУВАЛЬНИК ЗАВДАНЬ З СУЧАСНИМ ІНТЕРФЕЙСОМ
   - Створення завдань з кольоровими пріоритетами
   - Відстеження виконання з анімованими переходами
   - Нарахування очок з миттєвим зворотнім зв'язком
   - Інтерактивні елементи управління

6. ТЕСТИ ПРОДУКТИВНОСТІ З ВІЗУАЛЬНИМИ ЕФЕКТАМИ
   - Бенчмарк CPU з прогрес-барами
   - Тест швидкості RAM з неоновими індикаторами
   - Перевірка диска з динамічними результатами
   - Збереження результатів з анімацією

ДИЗАЙН 2025 РОКУ:
- Розмір вікна: 600x750 пікселів (адаптивний)
- Мінімальний розмір: 550x700 пікселів
- Колірна схема: градієнт від #0F0F0F до #2D2D2D
- Неонові акценти: #00DDEB (блакитний), #00FF66 (зелений)
- Шрифти: Arial/Roboto (оптимізовані для читабельності)
- Компактне відображення тексту з wraplength
- Ефекти наведення з плавними переходами
- Емодзі-іконки для кращого UX
- Статус бар з динамічними повідомленнями
- Оптимізовані розміри віджетів для кращого вміщення тексту

ТЕХНІЧНІ ДЕТАЛІ:
- Мова програмування: Python 3.11
- GUI фреймворк: Tkinter з custom стилями
- Системний моніторинг: psutil
- База даних: PostgreSQL + простий JSON backup
- Архітектура: модульна система з візуальними компонентами
- Сумісність: Windows, Linux, macOS

СТРУКТУРА ПРОЕКТУ:
main.py - головний файл запуску з класом TechCareApp
gui.py - сучасний графічний інтерфейс з неоновим дизайном 2025
monitor.py - збір системних даних у реальному часі
ai.py - власний алгоритм аналізу
repair.py - діагностика та автоматичний ремонт
achievements.py - система досягнень з очками
tests.py - тести продуктивності компонентів
data_manager.py - PostgreSQL менеджер
simple_data.py - альтернативний JSON менеджер

python main.py

Створення executable:
pip install pyinstaller
pyinstaller --onefile main.py
